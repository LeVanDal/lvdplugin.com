⚡ Tính năng

9 nhóm công cụ mạnh mẽ

Tất cả những gì kỹ sư thi công cần trong một plugin duy nhất. Từ giáo chống đến thép sàn, từ bóc tách đến shop drawing.

🧱

Ốp Lát 8 lệnh

Tạo pattern gạch tự động, chia gạch trong boundary, dim gạch, thống kê số lượng gạch và xuất BOM.

📐

Bóc tách Khối lượng 5 lệnh

Đo polyline, diện tích tự động. Xuất trực tiếp ra Excel hoặc bảng CAD. Hỗ trợ đo lường chính xác.

🏗️

Giáo chống 6 lệnh

Rải giáo tự động cho dầm giữa, dầm biên, sàn. Hỗ trợ 3 hệ giáo: Ringlock, Nêm và H-Frame. Tối ưu chiều cao tự động.

📏

Xà gồ 4 lệnh

Rải xà gồ 50×100 và 50×50 tự động vuông góc với tim dầm hoặc trong boundary sàn.

🔧

Tiện ích 8 lệnh

Tạo boundary, tìm tim dầm ảo thông minh (CBL), thống kê block, layout ván khuôn ply tự động, vẽ mặt cắt ván khuôn chi tiết.

⚙️

Thép sàn 4 lệnh

Vẽ thép sàn tự động/thủ công với 6 kiểu móc. Thống kê thép, xuất Excel với hình dạng, tạo shop drawing tự động.

🔩

Thép tăng cường 1 lệnh

Rải thép tăng cường hàng loạt. Tự động nhận diện mẫu, phân tích text thiết kế (d10a150), phân phối thép và tạo annotation.

📐

Nội suy cao độ 3 lệnh

Nội suy cao độ hoàn thiện tự động với 3 chế độ: Slope (độ dốc %), 2 điểm, và từ Line có sẵn trên bản vẽ.

🧱

Xây Tô 4 lệnh

Vẽ tường + lớp tô + dim + annotation tự động. Đục lỗ cửa đi / cửa sổ. Auto-dim kích thước tường xây tô. Shop Drawing mặt bằng định vị tường.

44+ lệnh chuyên nghiệp

Mỗi lệnh được tối ưu cho quy trình thi công thực tế, giúp bạn hoàn thành công việc nhanh gấp nhiều lần.

Xem kết quả thực tế

Các bản vẽ được tạo tự động bởi LVD Plugin trực tiếp trên AutoCAD.

Mặt cắt ván khuôn dầm

Mặt cắt ván khuôn sàn

Mặt cắt sàn chi tiết với 2 lớp xà gồ, hệ giáo chống và kích thước tự động.

Mặt bằng rải giáo dầm

Bố trí giáo chống Ringlock cho dầm BTCT với giằng dọc, giằng ngang và đĩa liên kết.

Mặt bằng rải giáo sàn

Bố trí giáo sàn Ringlock với giằng X, giằng Y, giằng phụ và đoạn cách biên tự động.

Hỗ trợ đa phiên bản AutoCAD

Plugin được build song song cho .NET 4.8 và .NET 8.0, đảm bảo tương thích từ AutoCAD 2021 đến 2025+.

🎯 Phiên bản AutoCAD

✓ AutoCAD 2021 .NET 4.8
✓ AutoCAD 2022 .NET 4.8
✓ AutoCAD 2023 .NET 4.8
✓ AutoCAD 2024 .NET 4.8
✓ AutoCAD 2025+ .NET 8.0

📥 Cài đặt nhanh

1
Tải file Setup (.exe) từ GitHub Releases
2
Chạy file cài đặt, chọn thư mục (mặc định tự động)
3
Mở AutoCAD — Plugin tự động load qua .bundle
4
Gõ lệnh LVD để mở bảng công cụ chính

Giải đáp thắc mắc

LVD Plugin hỗ trợ AutoCAD từ phiên bản 2021 trở lên. Phiên bản 2021-2024 sử dụng .NET Framework 4.8, còn từ 2025 trở đi sử dụng .NET 8.0. Plugin được build song song để đảm bảo tương thích.
License được gắn theo máy tính (theo serial ổ cứng) và có thời hạn sử dụng. Bạn sẽ được cung cấp key kích hoạt sau khi thanh toán. Key chỉ dùng được trên 1 máy duy nhất.
Có! Plugin sẽ tự động kiểm tra phiên bản mới khi khởi động AutoCAD. Nếu có bản cập nhật, bạn sẽ nhận được thông báo và link tải trực tiếp từ GitHub.
LVD Plugin hỗ trợ 3 hệ giáo chống phổ biến nhất: Ringlock (giáo đĩa), Nêm (giáo nêm), và H-Frame (giáo khung). Chiều cao giáo được tối ưu tự động dựa trên chiều cao tầng thực tế.
License gắn theo máy nên nếu bạn đổi máy, hãy liên hệ trực tiếp qua Zalo để được cấp key mới miễn phí (trong thời hạn license còn hiệu lực).
Bạn có thể thanh toán qua chuyển khoản ngân hàng (ACB). Sau khi thanh toán, gửi thông tin máy qua Zalo để nhận key kích hoạt trong vòng vài phút.

Viết LSP & Tools theo yêu cầu

Ngoài plugin có sẵn, chúng tôi nhận viết AutoLISP (LSP), Plugin C#, và các công cụ riêng lẻ theo nhu cầu cụ thể của dự án bạn.

📝

Viết AutoLISP (LSP)

Tự động hóa các thao tác lặp lại trên AutoCAD bằng LSP. Phù hợp cho các nhiệm vụ đơn giản đến trung bình, triển khai nhanh.

  • ✓ Tự động vẽ, chỉnh sửa bản vẽ
  • ✓ Xử lý text, block, layer hàng loạt
  • ✓ Tính toán, thống kê tự động
⚙️

Plugin C# cho AutoCAD

Phát triển plugin chuyên sâu bằng C# với giao diện WPF hiện đại. Hỗ trợ xử lý phức tạp, kết nối database, xuất báo cáo.

  • ✓ Giao diện WPF/WinForms chuyên nghiệp
  • ✓ Xuất Excel, kết nối cơ sở dữ liệu
  • ✓ Hỗ trợ AutoCAD 2021-2025+
📀

Tools riêng lẻ

Xây dựng công cụ độc lập (Excel macro, phần mềm Windows) phục vụ quy trình làm việc riêng của đội ngũ bạn.

  • ✓ Excel VBA / Macro tự động
  • ✓ Ứng dụng Windows Forms / WPF
  • ✓ Tư vấn giải pháp kỹ thuật
